The Files for the project where:

magentoExercise00/src/com/magento/exercise/MagentoRESTService.java
Service for get, transform, validate and send Email for Text incoming. The api is "/api/MagentoService"

magentoExercise00/src/com/magento/exercise/MagentoRESTServiceClient.java
Client for call the Service

magentoExercise00/Json/entryJson.txt
Json for exercice

/magentoExercise00/src/com/magento/domain:
Object Line, MagentoOrder, Order, ResponseMagento

magentoExercise00/src/com/magento/util/EmailUtil
Function for sendEmail

For run this exercises needs to Run Tomcat 7 for publish the Service (MagentoRESTService.java) and Run as application de client  (MagentoRESTServiceClient.java)
